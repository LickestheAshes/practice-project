ИНСТРУКЦИЯ ПО УСТАНОВКЕ И ЗАПУСКУ КУСКА ОМНИССИИ

1. Установите XAMPP (если ещё не установлен)

2. Скопируйте папку "korochki" в C:\xampp\htdocs\ (создайте предварительно)

3. Запустите Apache и MySQL в XAMPP Control Panel

4. Откройте phpMyAdmin (http://localhost/phpmyadmin)

5. Создайте базу данных:
   - Нажмите "Импорт"
   - Выберите файл "korochki_db.sql"
   - Нажмите "Вперёд"

6. Откройте сайт: http://localhost/korochki/

7. Для входа в админку:
   - Зарегистрируйтесь
   - В phpMyAdmin в таблице "user" измените роль на "admin"
   - Войдите снова
